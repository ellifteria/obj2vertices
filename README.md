# obj2vertices

## Usage

```text
Usage of obj2vertices:
  -input string
        the path to the .obj input file
  -ncoord int
        the number of coordinates in the output vertex list; all coordinates after X, Y, Z are automatically filled with 1.0; must be >= 3 (default 4)
  -output string
        the path to the vertex list output file (default "out/vertices.obj.txt")
```
